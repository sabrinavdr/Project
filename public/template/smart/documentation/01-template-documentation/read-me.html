3CSS Structure

This template includes 5 css files. Let's take a look closely at our stylesheet files:

  • style.css - common css styles for this template.
  • bootstrap.min.css - standard bootstrap css styles.
  • bootstrap-icons.css - contain css styles for bootstrap icons.
  • style-magnific-popup - contain css styles for the lightbox gallery.
  • animate.css- a cross-browser library of CSS animations

4JavaScripts

This template imports Javascript files. jQuery is a Javascript library that greatly reduces the amount of code that you must write. All the Javascript files can be found inside the js folder.

5Additional Info

Social Icons - please visit Font Awesome Icons Library and Stroke Icons Font for the complete list of social icons.

Fonts Used - We have used 2 type of Google Font in this template: Roboto and Roboto Slab. If you would like to use different Google Fonts, you have to copy the font link in Google Fonts page and paste it in your index.html file.

Then you will need to add the new font name to your style.css file.
